# MundorumFabulae.SDL3.Runtime

This package contains prebuilt SDL3 binaries for use in .NET applications.

## Included Platforms

The binaries included in this package are taken directly from the official SDL3 releases in their
[GitHub repository](https://github.com/libsdl-org/SDL/releases). As such, only the following platforms are included in
this package:

- Windows (x86, x64, ARM64)
- macOS (Universal/Universal2 - taken from `/SDL3/SDL3.xcframework/macos-arm64_x86_64/SDL3.framework/Versions/A/SDL3`)
